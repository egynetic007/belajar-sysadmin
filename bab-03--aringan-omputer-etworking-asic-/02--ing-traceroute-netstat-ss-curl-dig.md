# Ping, traceroute, netstat, ss, curl, dig

- PING (Packet Internet Groper)
  - adalah utilitas baris perintah yang digunakan untuk menguji konektivitas jaringan antara dua host. Ini bekerja dengan mengirimkan paket Internet Control Message Protocol (ICMP) "echo request" ke host target dan menunggu balasan "echo reply".
  ping google.con / ping 8.8.8.8
